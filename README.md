A full-featured hotel management web application built using Flask and MySQL.  
This system allows guests to browse rooms, make reservations, create accounts, and manage their bookings.  
Administrators can securely log in to monitor and manage hotel operations.

## Features

### Guest Features
- Room showcase with images and videos
- Real-time booking system with automatic price calculation
- User account creation and login
- User dashboard for managing bookings
- Fully responsive design

### Admin Features
- Secure admin login page
- Admin dashboard showing all customer bookings
- Ability to delete bookings
- Real-time monitoring of activity
